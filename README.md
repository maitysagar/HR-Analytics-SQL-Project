# HR Analytics SQL Project

## Objective
Analyze employee data to understand salary distribution,
department-wise trends, and identify high-paid employees.

## Database
MySQL 8.0

## Tools Used
- MySQL
- SQL

## Business Questions
1. How many employees are there in each department?
2. What is the average salary per department?
3. Which employees earn more than their department average?

## Key Learnings
- Used GROUP BY and aggregate functions
- Implemented correlated subqueries
- Used window functions for department-level analysis
- Converted SQL outputs into business insights
